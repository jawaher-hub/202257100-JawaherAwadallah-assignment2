# Portfolio Website (Assignment 2)

## Project Description
This is an interactive portfolio website showcasing my projects, and contact information. Built with HTML, CSS, and JavaScript.

## Features
- Dark/Light mode with persistence
- Project filtering by category
- Contact form

## Setup Instructions
1. Clone this repository
2. Open `index.html` in a web browser
## How to Interact with the Site
- Theme Toggle: Use the 🌙/☀️ icon in the top right to switch modes. Your preference is automatically saved for your next visit.

- Project Filters: Click the category buttons (All, Design, Technical) to instantly filter the projects.

- Contact Form: Fill out the form to see realtime validation. Upon submission, you will receive a "Sending..." status followed by a success confirmation. And the name you fill in the form is the name used to welcome you in the next visit.
